## 概览

Omnara REST API 提供对平台所有功能的编程访问。你可以用它来构建自定义智能体集成、仪表板或自动化工作流。

**基础 URL：** `https://agent.omnara.com/api/v1`

## 认证

所有 API 请求都需要通过 Bearer 令牌进行认证：

```bash
curl -H "Authorization: Bearer YOUR_API_KEY" \
  https://agent.omnara.com/api/v1/messages/agent
```

## 快速开始

从你的智能体发送一条消息：

```bash
curl -X POST https://agent.omnara.com/api/v1/messages/agent \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "agent_type": "My Custom Agent",
    "content": "Starting analysis of codebase",
    "requires_user_input": false
  }'
```

响应：

```json
{
  "success": true,
  "message_id": "msg_abc123",
  "agent_instance_id": "inst_xyz789",
  "queued_user_messages": []
}
```

## Python SDK

对于 Python 应用，请使用官方 SDK：

```bash
pip install omnara
```

```python
from omnara import OmnaraClient
import uuid

client = OmnaraClient(api_key="your-api-key")
instance_id = str(uuid.uuid4())

# 发送一条消息
response = client.send_message(
    agent_type="My Agent",
    content="Processing data...",
    agent_instance_id=instance_id,
    requires_user_input=False
)

# 提出一个问题
answer = client.send_message(
    content="Should I proceed with deployment?",
    agent_instance_id=instance_id,
    requires_user_input=True
)
print(f"User said: {answer}")
```

## 速率限制

当前的速率限制（可能会调整）：

| 级别 | 每分钟请求数 | 每小时请求数 |
|------|----------------|---------------|
| 免费版 | 60 | 1,000 |
| 专业版 | 300 | 10,000 |
| 企业版 | 自定义 | 自定义 |

响应中会包含速率限制相关的响应头：

```
X-RateLimit-Limit: 60
X-RateLimit-Remaining: 45
X-RateLimit-Reset: 1640000000
```

## 错误处理

API 使用标准 HTTP 状态码：

| 状态码 | 含义 |
|------|---------|
| 200 | 成功 |
| 400 | 错误请求——参数无效 |
| 401 | 未授权——API key 无效或缺失 |
| 403 | 禁止访问——权限不足 |
| 404 | 未找到——资源不存在 |
| 429 | 请求过多——超出速率限制 |
| 500 | 服务器内部错误 |

错误响应格式：

```json
{
  "error": {
    "code": "invalid_request",
    "message": "Missing required field: agent_type",
    "details": {
      "field": "agent_type"
    }
  }
}
```

## Webhook

配置 webhook，以便在用户回复时接收实时通知：

1. 在仪表板中设置 webhook URL
2. 在消息元数据中包含 webhook URL：

```json
{
  "agent_type": "My Agent",
  "content": "Approve deployment?",
  "requires_user_input": true,
  "message_metadata": {
    "webhook_url": "https://your-app.com/webhook",
    "webhook_type": "custom"
  }
}
```

3. 用户回复时接收 POST 请求：

```json
{
  "user_message": "Yes, approved",
  "user_id": "user_abc123",
  "message_id": "msg_xyz789",
  "agent_instance_id": "inst_def456",
  "timestamp": "2025-10-08T12:00:00Z"
}
```

## 自托管实例

指向你自己的 Omnara 实例：

```bash
export OMNARA_API_URL="https://omnara.mycompany.com"
```

或在 API 调用中指定：

```python
client = OmnaraClient(
    api_key="your-key",
    base_url="https://omnara.mycompany.com"
)
```
