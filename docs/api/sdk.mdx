## 安装

```bash
pip install omnara
```

SDK 包含在主 `omnara` 包中。

## 快速开始

```python
from omnara import OmnaraClient
import uuid

# 初始化客户端
client = OmnaraClient(api_key="your-api-key")

# 生成会话 ID
instance_id = str(uuid.uuid4())

# 发送状态更新
response = client.send_message(
    agent_type="My Custom Agent",
    content="Analyzing codebase structure",
    agent_instance_id=instance_id,
    requires_user_input=False
)

print(f"Message sent: {response['message_id']}")

# 请求用户输入
answer = client.send_message(
    content="Should I refactor this legacy module?",
    agent_instance_id=instance_id,
    requires_user_input=True
)

print(f"User answered: {answer}")
```

## 客户端初始化

<ParamField path="api_key" type="string" required>
  你的 Omnara API key
</ParamField>

<ParamField path="base_url" type="string" default="https://agent.omnara.com">
  API 服务器 URL（用于自托管实例）
</ParamField>

```python
from omnara import OmnaraClient

# 基本用法
client = OmnaraClient(api_key="xxxxxxxxxxxxxxxxxxxx")

# 自托管
client = OmnaraClient(
    api_key="xxxxxxxxxxxxxxxxxxxx",
    base_url="https://omnara.mycompany.com"
)

# 从环境变量读取
import os
client = OmnaraClient(api_key=os.getenv("OMNARA_API_KEY"))
```

## 发送消息

### 非阻塞消息

发送状态更新而无需等待：

```python
response = client.send_message(
    agent_type="Code Analyzer",
    content="Found 5 potential issues",
    agent_instance_id=instance_id,
    requires_user_input=False,
    send_push=True  # 发送推送通知
)

# 检查排队中的用户消息
if response.get("queued_user_messages"):
    for msg in response["queued_user_messages"]:
        print(f"User said: {msg['content']}")
```

### 阻塞式提问

提出问题并等待回复：

```python
answer = client.send_message(
    content="Which file should I start with?",
    agent_instance_id=instance_id,
    requires_user_input=True,
    timeout_minutes=60,  # 最多等待 1 小时
    poll_interval=10.0   # 每 10 秒检查一次
)

print(f"User selected: {answer}")
```

## 异步客户端

用于 async/await 场景：

```python
from omnara import AsyncOmnaraClient
import asyncio

async def main():
    client = AsyncOmnaraClient(api_key="your-api-key")

    # 发送消息
    response = await client.send_message(
        agent_type="Async Agent",
        content="Processing...",
        requires_user_input=False
    )

    # 提出问题
    answer = await client.send_message(
        content="Proceed?",
        agent_instance_id=response["agent_instance_id"],
        requires_user_input=True
    )

    print(answer)

asyncio.run(main())
```

## 会话管理

### 创建会话

会话会在发送第一条消息时自动创建：

```python
import uuid

# 生成唯一的会话 ID
session_id = str(uuid.uuid4())

# 第一条消息会创建会话
client.send_message(
    agent_type="My Agent",
    content="Starting work",
    agent_instance_id=session_id,
    requires_user_input=False
)
```

### 结束会话

将会话标记为已完成：

```python
client.end_session(agent_instance_id=session_id)
```

这会：
- 将状态设置为 COMPLETED
- 记录结束时间戳
- 停止将其作为活跃会话跟踪
- 在历史记录中保留该会话

## 高级功能

### Git Diff 跟踪

在消息中附带代码变更：

```python
import subprocess

def get_git_diff():
    result = subprocess.run(
        ["git", "diff", "HEAD"],
        capture_output=True,
        text=True
    )
    return result.stdout

# 携带 diff 发送
response = client.send_message(
    agent_type="Code Agent",
    content="Modified authentication flow",
    agent_instance_id=instance_id,
    requires_user_input=False,
    git_diff=get_git_diff()
)
```

diff 会显示在仪表板中，方便进行代码审查。

### 消息元数据

为消息附加自定义元数据：

```python
response = client.send_message(
    agent_type="Test Runner",
    content="Tests completed",
    agent_instance_id=instance_id,
    requires_user_input=False,
    message_metadata={
        "test_results": {
            "passed": 42,
            "failed": 3,
            "skipped": 1
        },
        "duration_ms": 1234,
        "commit_hash": "abc123def"
    }
)
```

元数据会随消息一起存储，并在仪表板中可见。

## 错误处理

```python
from omnara import OmnaraClient
from omnara.exceptions import OmnaraAPIError

client = OmnaraClient(api_key="your-key")

try:
    response = client.send_message(
        agent_type="My Agent",
        content="Test message",
        requires_user_input=False
    )
except OmnaraAPIError as e:
    print(f"API error: {e.status_code} - {e.message}")
    # 妥善处理错误
except Exception as e:
    print(f"Unexpected error: {e}")
```

## 完整示例

带有错误处理的完整智能体实现：

```python
from omnara import OmnaraClient
import uuid
import time

def run_agent():
    # 初始化
    client = OmnaraClient(api_key="your-key")
    session_id = str(uuid.uuid4())

    try:
        # 开始会话
        client.send_message(
            agent_type="Code Review Agent",
            content="Starting code review",
            agent_instance_id=session_id,
            requires_user_input=False
        )

        # 执行一些工作
        time.sleep(2)

        # 汇报进度
        client.send_message(
            content="Analyzed 10 files, found 3 issues",
            agent_instance_id=session_id,
            requires_user_input=False
        )

        # 请求指示
        answer = client.send_message(
            content="Should I auto-fix the linting issues?",
            agent_instance_id=session_id,
            requires_user_input=True,
            timeout_minutes=30
        )

        # 处理回复
        if "yes" in answer.lower():
            client.send_message(
                content="Fixing issues...",
                agent_instance_id=session_id,
                requires_user_input=False
            )
            # 修复问题...
            time.sleep(5)
            client.send_message(
                content="All issues fixed! ✅",
                agent_instance_id=session_id,
                requires_user_input=False,
                send_push=True
            )
        else:
            client.send_message(
                content="Skipping auto-fix as requested",
                agent_instance_id=session_id,
                requires_user_input=False
            )

    finally:
        # 始终结束会话
        client.end_session(agent_instance_id=session_id)

if __name__ == "__main__":
    run_agent()
```

## SDK 参考

### OmnaraClient

**方法：**

<ParamField path="send_message" type="function">
  发送一条消息或提出一个问题

  **参数：**
  - `agent_type` (str, optional)：智能体类型名称
  - `content` (str, required)：消息内容
  - `agent_instance_id` (str, optional)：会话 ID
  - `requires_user_input` (bool, default=False)：是否等待回复
  - `timeout_minutes` (int, default=1440)：等待超时时间（24 小时）
  - `poll_interval` (float, default=10.0)：轮询间隔（秒）
  - `send_email` (bool, default=False)：发送邮件通知
  - `send_sms` (bool, default=False)：发送短信通知
  - `send_push` (bool, default=False)：发送推送通知
  - `git_diff` (str, optional)：Git diff 内容
  - `message_metadata` (dict, optional)：自定义元数据

  **返回值：**
  - 当 `requires_user_input=False` 时：包含 `message_id`、`agent_instance_id`、`queued_user_messages` 的响应字典
  - 当 `requires_user_input=True` 时：用户的回复文本（字符串）
</ParamField>

<ParamField path="end_session" type="function">
  将会话标记为已完成

  **参数：**
  - `agent_instance_id` (str, required)：要结束的会话

  **返回值：** 表示成功的响应字典
</ParamField>

### AsyncOmnaraClient

接口与 `OmnaraClient` 相同，但使用 async/await：

```python
from omnara import AsyncOmnaraClient

async def async_agent():
    client = AsyncOmnaraClient(api_key="your-key")

    response = await client.send_message(
        agent_type="Async Agent",
        content="Processing async task",
        requires_user_input=False
    )

    answer = await client.send_message(
        content="Continue processing?",
        agent_instance_id=response["agent_instance_id"],
        requires_user_input=True
    )

    await client.end_session(agent_instance_id=response["agent_instance_id"])
```
