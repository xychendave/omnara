## 可用智能体

Omnara 支持三种 AI 编码智能体：

<CardGroup cols={3}>
  <Card title="Claude Code" icon="terminal">
    **默认智能体**

    Anthropic 的 Claude Code CLI，提供完整的集成支持
  </Card>
  <Card title="Codex" icon="code">
    **基于 Rust**

    使用 Rust 构建的快速、原生 Omnara 集成
  </Card>
  <Card title="Amp" icon="wand-magic-sparkles">
    **高级解析**

    精细的终端输出解析能力
  </Card>
</CardGroup>

## 设置默认智能体

保存你偏好的智能体，避免每次都指定 `--agent`：

<CodeGroup>

```bash Set and Exit
# 设置默认智能体并退出
omnara --set-default codex
```

```bash Set and Launch
# 设置默认值并立即启动
omnara --agent amp --set-default
```

</CodeGroup>

默认值存储在 `~/.omnara/config.json` 中，并用于之后的所有会话。

## 切换智能体

临时覆盖默认智能体：

```bash
# 使用 Claude Code（即使 codex 是默认值）
omnara --agent claude

# 以自定义名称使用 Codex
omnara --agent codex --name "Backend Work"
```

## 智能体特有功能

<Tabs>
  <Tab title="Claude Code">
    ### 功能
    - 将权限提示转发到仪表盘
    - 按消息跟踪 git diff
    - 会话日志解析（JSONL 格式）
    - JSONL 消息流式传输
    - 支持计划模式

    ### 要求
    ```bash
    # 验证 Claude Code 是否已安装
    claude --version
    ```

    ### 配置
    通过 Omnara 使用时，支持所有标准的 Claude Code 标志。
  </Tab>

  <Tab title="Codex">
    ### 功能
    - 原生 Omnara 集成（内置）
    - 终端流式传输
    - 会话管理
    - 心跳跟踪
    - 基于 Rust 的快速实现

    ### 要求
    随 Omnara 一起打包——无需单独安装。

    ### 二进制文件位置
    你可以覆盖二进制文件路径：
    ```bash
    export OMNARA_CODEX_PATH=/path/to/custom/codex
    ```
  </Tab>

  <Tab title="Amp">
    ### 功能
    - 区分思考与回复内容
    - ANSI 码解析
    - 支持 TUI 交互
    - 高级输出过滤

    ### 要求
    ```bash
    # 验证 Amp 是否已安装
    amp --version
    ```

    ### 配置
    通过 Omnara 运行时，Amp 会使用 `--dangerously-allow-all` 自动绕过权限检查。
  </Tab>
</Tabs>

## 配置文件

你的默认智能体偏好存储在 `~/.omnara/config.json` 中：

```json
{
  "default_agent": "codex"
}
```

你可以手动编辑此文件，或使用 `--set-default` 标志。

## 智能体选择优先级

当你运行 `omnara` 时，智能体按以下顺序选择：

1. **显式的 `--agent` 标志**（最高优先级）
2. **`~/.omnara/config.json` 中的默认值**
3. **硬编码默认值：`claude`**（在没有配置文件时）

## 智能体显示名称

自定义你的智能体在仪表盘中的显示方式：

```bash
# 设置一个描述性名称
omnara --agent codex --name "Feature Development"

# 使用环境变量
export OMNARA_AGENT_DISPLAY_NAME="Code Review Bot"
omnara
```

当你同时运行多个智能体时，显示名称有助于识别各个会话。
