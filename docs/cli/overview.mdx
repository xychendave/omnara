## 快速示例

<CodeGroup>

```bash Claude Code
# 启动带有仪表盘集成的 Claude Code
omnara
```

```bash Codex
# 以自定义名称启动 Codex
omnara --agent codex --name "Backend Refactor"
```

```bash Headless
# 以后台模式运行 Claude Code 并自动接受编辑
omnara headless --permission-mode acceptEdits
```

```bash Webhook Server
# 启动带有公开 URL 的 webhook 服务器
omnara serve
```

</CodeGroup>

## 全局选项

以下选项适用于所有命令：

| 选项 | 描述 |
|--------|-------------|
| `--api-key <key>` | 用于身份验证的 API key |
| `--auth` | 进行身份验证或重新验证 |
| `--reauth` | 强制重新进行身份验证 |
| `--version` | 显示版本信息 |
| `--base-url <url>` | 自定义 API 服务器 URL |
