## 概述

Omnara 的 GitHub Actions 集成允许你在 GitHub Actions 工作流中触发 Claude Code 运行，同时通过 Omnara 仪表板监控并与其交互。

## 快速开始

<Steps>
  <Step title="添加 GitHub Action">
    创建 `.github/workflows/omnara.yml`：

    ```yaml
    name: Omnara Claude Code
    on:
      repository_dispatch:
        types: [omnara-trigger]

    jobs:
      claude:
        runs-on: ubuntu-latest
        steps:
          - uses: actions/checkout@v3

          - name: Run Claude Code
            uses: omnara-ai/claude-code-action@v1
            with:
              api-key: ${{ secrets.OMNARA_API_KEY }}
              prompt: ${{ github.event.client_payload.prompt }}
              permission-mode: acceptEdits
    ```
  </Step>

  <Step title="配置仪表板">
    1. 在你的[仪表板](https://claude.omnara.com/dashboard)中进入 **Settings → Integrations**
    2. 添加 GitHub Actions 集成
    3. 输入仓库名称（例如 `owner/repo`）
    4. 添加具有 `repo` 权限范围的 GitHub token
  </Step>

  <Step title="从仪表板触发">
    点击「Launch Agent」并选择 GitHub Actions，即可在你的仓库中触发 Claude Code
  </Step>
</Steps>

## 配置

### Action 输入参数

<ParamField path="api-key" type="string" required>
  用于身份验证的 Omnara API key
</ParamField>

<ParamField path="prompt" type="string">
  Claude Code 的初始提示词
</ParamField>

<ParamField path="permission-mode" type="string">
  权限处理方式：`acceptEdits`、`bypassPermissions`、`plan`、`default`
</ParamField>

<ParamField path="allowed-tools" type="string">
  允许使用的工具列表（以逗号分隔）
</ParamField>

<ParamField path="branch" type="string">
  要工作的分支（默认：创建新分支）
</ParamField>

### 必需的 Secrets

将以下内容添加到你的仓库 secrets 中：

```bash
OMNARA_API_KEY  # 从 omnara.com/dashboard 获取
GITHUB_TOKEN    # 由 GitHub Actions 自动提供
```

## 工作流示例

### 在 Issue 评论时触发

```yaml
name: Claude Code on Demand
on:
  issue_comment:
    types: [created]

jobs:
  claude:
    if: contains(github.event.comment.body, '/claude')
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3

      - name: Extract Command
        id: command
        run: |
          COMMAND=$(echo "${{ github.event.comment.body }}" | sed 's/\/claude //')
          echo "prompt=$COMMAND" >> $GITHUB_OUTPUT

      - name: Run Claude
        uses: omnara-ai/claude-code-action@v1
        with:
          api-key: ${{ secrets.OMNARA_API_KEY }}
          prompt: ${{ steps.command.outputs.prompt }}
```

### 定时代码维护

```yaml
name: Weekly Maintenance
on:
  schedule:
    - cron: '0 0 * * 0'  # 每周日

jobs:
  maintenance:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3

      - name: Run Maintenance
        uses: omnara-ai/claude-code-action@v1
        with:
          api-key: ${{ secrets.OMNARA_API_KEY }}
          prompt: "Update dependencies and fix any linting issues"
          permission-mode: acceptEdits
```
