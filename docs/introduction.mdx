## 什么是 Omnara？

Omnara 让你的 AI 智能体（Claude Code、Codex CLI、n8n 等）从沉默的工作者变成善于沟通的队友。在网页端和移动端的统一仪表盘上，实时了解你的智能体正在做什么，并即时回复它们的提问。

## 核心功能

<AccordionGroup>
  <Accordion title="实时仪表盘" icon="gauge">
    在终端、网页和移动端实时查看你的 AI 智能体正在做什么。再也不会失去对智能体的掌控。
  </Accordion>
  <Accordion title="双向沟通" icon="messages">
    你的智能体可以提出问题并等待你的回复。无论身在何处，都可以通过手机、网页仪表盘或终端进行回复。
  </Accordion>
  <Accordion title="多智能体支持" icon="robot">
    支持 Claude Code、Codex CLI、n8n 工作流，并可通过我们的 SDK 或 REST API 接入任意自定义智能体。
  </Accordion>
  <Accordion title="会话持久化" icon="database">
    所有消息和会话都被安全存储。可以恢复对话、查看历史记录，并随时间追踪进展。
  </Accordion>
</AccordionGroup>

## 快速开始

一分钟内即可上手：

<CodeGroup>

```bash pip
pip install omnara
omnara
```

```bash homebrew
brew install omnara
omnara
```

```bash uv
uv tool install omnara
omnara
```

</CodeGroup>

就这么简单！按提示创建账户，然后在终端中与 Claude 交互。所有内容都会自动同步到你的[仪表盘](https://claude.omnara.com/dashboard)。
