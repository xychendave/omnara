<Steps>
  <Step title="安装 Omnara">
    选择你偏好的包管理器：

    <CodeGroup>
    ```bash pip
    pip install omnara
    ```

    ```bash homebrew
    brew install omnara
    ```

    ```bash uv
    uv tool install omnara
    ```

    ```bash pipx
    pipx install omnara
    ```
    </CodeGroup>
  </Step>

  <Step title="启动你的智能体">
    启动带仪表盘集成的 Claude Code：

    ```bash
    omnara
    ```

    首次运行时，它将：
    1. 打开浏览器进行认证
    2. 提示你登录 Omnara（或创建账户）
    3. 自动将你的 API key 保存到 `~/.omnara/credentials.json`

    你在终端中的所有操作现在都会同步到你的[仪表盘](https://claude.omnara.com/dashboard)和移动应用。

    <Tip>
    首次运行时会自动进行认证。如有需要，也可以使用 `omnara --auth` 显式认证。
    </Tip>
  </Step>

  <Step title="尝试其他模式">
    探索不同的使用方式：

    <CodeGroup>
    ```bash Headless
    # 在后台运行，无终端 UI
    omnara headless --prompt "Review this codebase"
    ```

    ```bash Webhook Server
    # 启用远程触发
    omnara serve
    ```

    ```bash Different Agent
    # 使用 Codex 替代 Claude Code
    omnara --agent codex
    ```
    </CodeGroup>
  </Step>
</Steps>

## 常见工作流

<AccordionGroup>
  <Accordion title="使用 Claude Code 进行开发" icon="code">
    ```bash
    # 将 Claude 设为默认
    omnara --set-default claude

    # 启动一个命名会话
    omnara --name "Feature Development"

    # 像往常一样在终端中工作
    # 所有内容自动同步到仪表盘
    ```
  </Accordion>

  <Accordion title="后台代码审查" icon="magnifying-glass">
    ```bash
    # 启动 headless 审查
    omnara headless \
      --prompt "Review all files in src/ for security issues" \
      --allowed-tools Read,Grep \
      --cwd /path/to/project

    # 在仪表盘查看结果
    # 回复 Claude 提出的任何问题
    ```
  </Accordion>

  <Accordion title="远程移动开发" icon="mobile">
    ```bash
    # 在你的电脑上，启动 webhook 服务器
    omnara serve

    # 在你的手机上：
    # 1. 打开 Omnara 应用
    # 2. 进入 Settings → Integrations
    # 3. 添加 webhook 配置
    # 4. 点击 "Launch Agent" 在你的电脑上触发 Claude
    ```
  </Accordion>
</AccordionGroup>

## 故障排查

<AccordionGroup>
  <Accordion title="认证失败">
    ```bash
    # 尝试重新认证
    omnara --reauth

    # 或手动设置 API key
    export OMNARA_API_KEY="your-key-here"
    omnara
    ```
  </Accordion>

  <Accordion title="找不到智能体">
    确保智能体的 CLI 已安装：

    ```bash
    # Claude Code
    claude --version

    # Codex（已随 Omnara 打包）
    # 应当开箱即用

    # Amp
    amp --version
    ```
  </Accordion>

  <Accordion title="仪表盘未更新">
    1. 检查你是否使用了 `--no-relay`
    2. 确认网络连接正常
    3. 检查日志：`~/.omnara/<agent>_wrapper/<session-id>.log`
  </Accordion>
</AccordionGroup>
